# Personal Portfolio Website

Welcome to the repository for my **Personal Portfolio Website**!  
This project is designed to showcase my skills, projects, and professional profile using modern web development practices.

## 📂 Project Overview

This is a responsive and visually appealing personal portfolio website created using:
- **HTML** for structure.
- **CSS** for styling.
- **JavaScript** for interactivity.
- **Bootstrap** for responsive design.

## 🖥️ Features

- **Home Page**: Introduction, professional summary, and links to other sections.
- **Inner Page**: Additional content or information pages.
- **Portfolio Details Page**: Showcases specific project details with images and descriptions.
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.

## 📄 Project Files

| File Name               | Description                          |
|-------------------------|--------------------------------------|
| `index.html`            | Main homepage of the portfolio.     |
| `inner-page.html`       | Secondary content page.             |
| `portfolio-details.html`| Details about specific projects.    |
